# QQ Email Outreach Skill

This repository provides a downloadable Codex skill package for QQ Mail outreach workflows.

## Download

Download `qq-email-outreach-skill.zip` from this repository, unzip it, and copy the `qq-email-outreach` folder into:

```text
C:\Users\<your-user-name>\.codex\skills\
```

Restart Codex after copying the folder.

## Usage

Ask Codex:

```text
Use qq-email-outreach to initialize a QQ email outreach workspace.
```

The skill can help preview recipients, send one test email, diagnose QQ SMTP connectivity, and send a confirmed small campaign batch.

## Safety

Do not paste QQ passwords or authorization codes into chat. Configure them locally in the generated `.env` file.
